## 🎯 COMPLETED TODAY (Mar 1, 2026)

### Frontend Bug Fixes - v3.2.1

**Goal:** Fix critical frontend issues affecting user experience
**Status:** ✅ COMPLETE

**Issues Fixed:**

1. **SES/React Bundling Issue**
   - Problem: MetaMask injects SES which removes React intrinsics (useEffect, useState)
   - Solution: Added polyfills in index.html that run BEFORE the module loads
   - Files: `tais_frontend/index.html`

2. **Wallet Session Restoration**
   - Problem: Wallet session wasn't restoring on page load - users had to reconnect
   - Solution: Added useEffect in useWallet.ts to restore session from JWT token
   - Files: `tais_frontend/src/hooks/useWallet.ts`

3. **Local Backup/Restore Signer Access**
   - Problem: Using `wallet.signer` which doesn't exist in useWallet hook
   - Solution: Get signer directly from ethers provider: `new providers.Web3Provider(window.ethereum).getSigner()`
   - Files: `tais_frontend/src/app/components/memory/MemoryArchivePage.tsx`

4. **OAuth getPermissions Error**
   - Problem: Using `require('../../services/oauthApi')` was causing issues
   - Solution: Changed to proper ES import in Dashboard
   - Files: `tais_frontend/src/app/components/Dashboard.tsx`

5. **OpenSea Metrics Removed**
   - Problem: Landing page was calling OpenSea API which wasn't working
   - Solution: Removed OpenSea metrics from landing page

6. **Renamed "AI Interview" to "Live Agent Console"**
   - Renamed "Guided Design" feature to "Live Agent Console"
   - Added agent status bar showing current agent state
   - Files: `tais_frontend/src/app/components/conversation/DynamicConversationContainer.tsx`

7. **TensorFlow Platform Warning**
   - Problem: Platform already set warning on initialization
   - Solution: Fixed platform detection in tensorflow.ts service

8. **Favicon Added**
   - Added favicon to the application

9. **Auto-load Agents on Wallet Connect**
   - Problem: Agents weren't loading when wallet connected
   - Solution: Added auto-load agents when wallet connects in Dashboard

10. **CTO Projects 400 Error**
    - Problem: Backend returning 400 errors on /api/v1/cto/projects
    - Solution: Frontend wasn't passing wallet address in API calls
    - Fixed loadProjects to pass wallet in query param
    - Fixed createProject to pass wallet in request body

11. **Agent Detail Modal Scope Issue**
    - Problem: showKnowledgePicker and showVersionHistory not accessible in nested AgentDetailModal
    - Solution: Pass all required state as props to AgentDetailModal
    - Files: `tais_frontend/src/app/components/Dashboard.tsx`

12. **RAG Auto-Initialization Causing Unwanted Signatures**
    - Problem: Public RAG was initializing on page load, requiring MetaMask signature
    - Solution: Removed auto-initialize, only initialize when user clicks Edit
    - Files: `tais_frontend/src/app/components/Dashboard.tsx`

13. **Monaco Editor Not Loading**
    - Problem: Monaco Editor failing to load from CDN, blocking agent editing
    - Solution: Replaced Monaco Editor with simple textarea
    - Files: `tais_frontend/src/app/components/Dashboard.tsx`

14. **TensorFlow.js Backend Initialization**
    - Problem: "No backend found in registry" error when uploading to Public RAG
    - Solution: Added ensureBackend() call before loading embedding model
    - Files: `tais_frontend/src/services/rag/embeddings.ts`

15. **CSP Headers for TensorFlow.js**
    - Problem: TensorFlow.js model loading blocked by CSP
    - Solution: Added tfhub.dev, storage.googleapis.com, www.kaggle.com to CSP
    - Files: `tais_frontend/vercel.json` (later removed as causing issues)

**npm Package:** `tais-rag-sdk@1.0.0`
**Install:** `npm install tais-rag-sdk`

**SDK Features Implemented:**
```typescript
import { TAISClient } from 'tais-rag-sdk';

const client = new TAISClient({ walletAddress: '0x...' });

// Wallet signature authentication (requires Gold tier / Genesis NFT)
const result = await client.authenticateWithSignature(async (message) => {
  return await wallet.signMessage(message);
});
console.log(result.apiKey, result.tier); // 'tais_sdk_...', 'gold'

// Upload encrypted document
const doc = await client.uploadDocument(encryptedData, {
  title: 'My Document',
  tags: ['ai', 'research'],
  isPublic: false
});

// Search documents
const results = await client.search(queryHash, { topK: 5 });

// Get quota
const quota = await client.getQuota();
```

**Auth Endpoints:**
- `GET /api/v1/sdk/auth/challenge?wallet=0x...` - Get challenge to sign
- `POST /api/v1/sdk/auth/authenticate` - Submit signature, get API key
- `GET /api/v1/sdk/auth/verify` - Verify API key is valid
- `DELETE /api/v1/sdk/auth/revoke` - Revoke API key

**Gold Tier Requirement:**
- Only Genesis NFT holders can authenticate
- Returns 403 with upgrade URL if not Gold tier
- API keys valid for 30 days
- Rate limit: 10,000 requests/day
